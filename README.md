# AutoIt CDP Language Server

Provides IntelliSense, hover information, signature help, and CDP‑based completions for AutoIt scripts.

Features:
- AutoIt syntax highlighting
- CDP schema‑driven completions for $cdp, $browser, $page, locators
- Hover tooltips for CDP methods
- Signature help for CDP commands
- AutoIt language configuration (comments, folding, brackets)

This extension is under active development.
